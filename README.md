## Descripción
Este proyecto contiene 10 ejercicios aplicando conceptos de:

- Herencia
- Sobrescritura de métodos (@Override)
- Uso de super
- Encapsulamiento

## Ejercicios incluidos
1. Persona, Docente, Estudiante
2. Cuenta y CuentaCorriente
3. Producto y ProductoPerecedero
4. Transporte y Bus
5. Empleado y Gerente
6. Animal y Perro
7. Vehiculo y Moto
8. Figura, Rectangulo y Circulo
9. Libro y LibroDigital
10. Sistema de Facturación

## Estructura
- src/herencia → contiene todas las clases del proyecto
